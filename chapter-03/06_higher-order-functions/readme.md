# 고차 함수
+ 다른 함수를 조작할 수 있는 함수
    + 다른 함수를 인자로 받을 수 있음
    + 다른 함수를 반환할 수 있음
+ 비동기적인 실행 맥락을 처리할 때 유용
+ 필요할 때 재활용할 수 있는 함수를 만들 수 있음

+ Array.map, Array.filter, Array.reduce
